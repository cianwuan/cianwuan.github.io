Moke 客户端，第三方最后的倔强

2019-11-29 4 min read # 随笔 # 科技

28号晚上，Moke 开发者推送了 Moke 3 的AppStore 更新，我也在第一时间进行下载使用。

使用下来发现，3.0 版本相较于 2.0 版本并没有做特别大的改进，更多的是在现有功能基础上进行调整和打磨，在 UI 视觉上适配 iOS 13 的特性。同时从 2.0 的买断制改为了订阅制，这引起了不少人的讨论。

Moke 是什么

Moke 是一款 iOS 上的第三方微博客户端，有人就有疑惑，为什么不用微博官方原生的客户端，免费的不挺好呀，干嘛还需要花钱去用第三方的软件。

答案就是官方客户端真的太难用了，列出几点我无法忍受的地方：

  • 广告，这是重点，太多广告，无处不在的广告和推广，太多！太多！太多！而且微博官方也是一个很奇葩的存在，微博会员居然没有去广告功能，赚着广告商的钱同时又在收割用户的钱包，这种吃相真的让我感到恶心。时间线上充斥着各种推广，太多干扰阅读的元素。
  • 混乱的时间线，强烈吐槽的，既然微博已经将其社交属性让道给媒体属性，那么媒体报道的时效性都无法得到保证，你说有算法可以优化显示顺序，但是就不能好好的从新到旧或者从旧到新的顺序让用户进行时间线的阅读吗？
  • 冗余的功能，官方客户端有太多功能我都需求不上，与其说是在阅读微博，更像是在用一个大型的多功能聚合软件，丝毫没有任何阅读的轻松感，同理，支付宝也是一个样子。
  • 混乱的 UI ，官方客户端将太多元素统统塞进去，想使用一个功能有时需要进好几级菜单才能找到。
  • ……

是的，我就是如此矫情，我就是受不了这以上些，你能脑补自动去广告我做不到，你能从混乱的时间线中提取你需要的信息我做不到……

于是在这种需求之下利用微博“开放” API ，诞生了许多第三方微博客户端。而第三方微博的发展也经历了好几个时期，由盛到衰，有机会可以细说这个有趣的现象。而现在市面上还存活着的客户端已经屈指可数了，而 Moke 就是这其中的佼佼者。

Moke 很好的解决了上述的痛点，只做最本分的事，那就是如何舒适地浏览微博。

关于订阅

“能用就可以了”和“好用”是两种思想，如果你是前者，那就出门右拐微博官方客户端。如果你是后者再继续讨论。

Moke 改为订阅制，对于我来说完全可以理解。Moke 2已经上架了5年了，当初买断价格是18元。这笔帐怎么算都是很值了，作为一个每天都使用的软件，折算下来一天所分摊的价格很少很少了。而订阅制，一个月7元或者一年73元，每天折算下来价格也是很低。很多人无法接受是在于并没有把观念转换过来。一个每天都经常使用的事物，说明它对于你来说，使用价值是非常高的。而一个有如此高使用价值的事物，是有偿的是理所应当的吧，这件事换成其他实物可能你就会理解了。这一个观点

“买断制”和“订阅制”本质上其实就是关于所有权和使用权的区别。但是如果你是一个佛教徒就会知道所谓的所有权其实是个伪命题。我们其实并没有真正意义上的拥有过任何事物。所以这一点对于我来说并没有带来太大心理障碍。
